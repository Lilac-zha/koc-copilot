## 1. 快速开始

### 1.1 KOC Copilot 是什么？

KOC Copilot 是专为 **1000-20000 粉丝量级内容创作者** 设计的 AI 成长助手。

如果你正在经历以下困境，这个工具正是为你而生：

- 🤔 **不知道发什么**：每天对着草稿发呆，不知道哪个选题最值得做
- 📉 **互动率下滑**：内容质量没变，但点赞评论越来越少
- 💸 **接广告担心掉粉**：想变现但怕观众反感，不知道如何平衡
- 🌊 **红海竞争激烈**：热门话题都有人做了，不知道自己的差异化在哪

KOC Copilot 通过分析你的**账号定位**，为你规划最优的内容策略——不是给所有人一样的建议，而是专属于你的个性化路径。

### 1.2 核心概念：信任资产

在 KOC Copilot 的体系里，你的账号有两种资产：

| 资产类型 | 说明 | 特点 |
|---------|------|------|
| **信任资产** | 粉丝对你的信任程度 | 慢慢积累，很难快速恢复 |
| **商业变现** | 广告/带货带来的收入 | 每次商业内容都会消耗信任资产 |

**核心规律：当商业内容超过总内容的 20%，信任资产开始折旧。一旦超过 40%，粉丝会大量流失，商业价值反而急速下降。**

KOC Copilot 帮你找到**信任积累与商业变现之间的最优平衡点**，让你实现长期可持续增长。

### 1.3 数据飞轮

```
输入账号信息
    ↓
AI 生成账号定位种子
    ↓
获得个性化选题推荐
    ↓
创作并发布内容
    ↓
数据回流更新种子（种子版本 v1 → v2 → v3...）
    ↓
推荐越来越精准（飞轮持续转动）
```

每发一篇内容、每一次数据反馈，系统的推荐都会更懂你。

## 2. 第一步：账号定位设置

### 2.1 选择设置路径

进入 KOC Copilot 后，你会看到两种设置路径：

#### 路径 A：导入历史内容（推荐）

适合已有内容积累的创作者。

**操作步骤：**
1. 点击「导入历史内容」
2. 将你最近发布的 **3-5 篇内容** 粘贴到文本框中
3. 包含：标题 + 正文（不需要数据，只需要内容本身）
4. 点击「AI 分析」，等待约 15 秒

**AI 会分析什么？**
- 你的内容垂类和细分方向
- 你的语言风格（偏理性/感性/幽默/专业）
- 你的核心价值主张
- 你最擅长的内容类型

**示例输入：**
```
标题：我如何用番茄工作法让工作效率提升 200%
正文：上班族最大的敌人不是工作量，而是注意力分散...

标题：深度工作 vs 浅度工作，99% 的人搞错了优先级
正文：Cal Newport 说得对，注意力是 21 世纪最稀缺的资源...
```

#### 路径 B：填写问卷（适合新账号）

适合刚起步或想重新定位的创作者。

**6 个关键问题：**

| 问题 | 说明 | 示例 |
|------|------|------|
| Q1 内容方向 | 你主要做哪个垂类的内容 | 职场效率、家居改造、宠物养护... |
| Q2 表达风格 | 你习惯用什么语气说话 | 理性分析型、温暖陪伴型... |
| Q3 目标观众 | 你的内容主要给谁看 | 25-35岁职场女性、宝妈群体... |
| Q4 最大困扰 | 你现在最想解决的问题（最多选 3 个）| 互动率低、不知道发什么... |
| Q5 账号状态 | 当前粉丝量 + 发布频率 | 滑动选择 |
| Q6 AI助力重点 | 你希望 AI 主要帮你做什么 | 选题灵感、内容打磨... |

> 💡 **提示**：每个问题都有「其他（填写）」选项，如果预设选项不适合你，请填写更精准的描述，系统会为你生成更个性化的结果。

### 2.2 查看账号定位档案

分析完成后，你会看到「AI 眼中的你」：

```
┌─────────────────────────────────────┐
│  📋 你的账号定位档案                  │
│                                     │
│  内容方向：职场效率                   │
│  语言风格：理性分析型                 │
│  目标观众：25-35岁职场人              │
│  内容优势：方法论总结、工具测评        │
│  语言习惯：用数据说话、给具体步骤      │
│                                     │
│  账号状态：2000-5000粉 | 成长期        │
│  信任健康度：65/100 ⚠️ 注意           │
│                                     │
│  种子版本：v1                         │
```

**如果不准确怎么办？** 直接修改对应字段，重新点击「更新定位」，系统会基于修改后的信息重新生成推荐。

## 3. 第二步：选题地图

选题地图是 KOC Copilot 最核心的功能——**用蓝海算法找到你专属的内容机会**。

### 3.1 蓝海分是什么？

```
蓝海分 = 话题热度 × (1 - 竞争程度) × 成长趋势系数

蓝海分 > 15 → 🔵 蓝海机会（做了大概率有流量）
蓝海分 10-15 → 🟡 适中（可以做）
蓝海分 < 10 → 🔴 红海竞争（慎入）
```

**举例理解：**
- 话题热度 60（有人搜，有需求）
- 竞争程度 0.3（竞争者少）
- 趋势：上升（× 1.3）
- 蓝海分 = 60 × 0.7 × 1.3 = **54.6 → 极佳蓝海**

相比之下：
- 话题热度 90（很多人搜）
- 竞争程度 0.9（满屏都是竞品）
- 蓝海分 = 90 × 0.1 × 1.0 = **9 → 红海竞争**

### 3.2 解读选题卡片

每个推荐选题都包含：

```
┌─────────────────────────────────────────┐
│ 📝 番茄工作法的进阶用法（很多人用错了）     │
│                                         │
│ 蓝海分：54.6 🔵          热度：↗ 上升     │
│                                         │
│ 为什么推荐给你：                          │
│ 你的「理性分析型」风格非常适合拆解方法      │
│ 论，当前该话题竞争者多为表面介绍，你可以    │
│ 做进阶+纠误类内容形成差异化。             │
│                                         │
│ 内容类型：常青内容 ✅（长期有搜索流量）      │
│ 预估互动率：3.2% - 4.8%                  │
│                                         │
│           [开始创作 →]                   │
└─────────────────────────────────────────┘
```

**各字段说明：**

| 字段 | 说明 |
|------|------|
| 蓝海分 | 机会值，越高越好 |
| 热度趋势 | ↗上升 / → 稳定 / ↘下降 |
| 为什么推荐给你 | AI 结合你的人设给出的专属理由 |
| 内容类型 | 常青（长效）/ 即时（蹭热点）/ 商业（变现） |
| 预估互动率 | 基于类似账号历史数据的区间估算 |

### 3.3 内容类型健康配比

KOC Copilot 推荐的黄金内容结构：

```
常青内容：70%  ←  长期搜索流量 + 信任积累
即时内容：20%  ←  蹭热点 + 涨新粉
商业内容：10%  ←  变现（不超过 20%）
```

> ⚠️ **注意**：选题地图会自动标注每个选题的类型，帮你控制商业内容占比不超过 20%。

### 3.4 刷新推荐

点击「重新推荐」可以获取新的一批选题。推荐结果每次都会略有不同，因为 AI 会在你的垂类范围内探索不同的角度。

## 4. 第三步：内容创作

### 4.1 三档创作模式

选定选题后，选择适合你今天状态的创作模式：

#### 🟢 轻模式（一键生成，直接发）

**适合场景：** 内容日历要求、日常维活、时间紧迫时

**AI 产出：**
- 完整内容草稿（开头钩子 + 正文 + 互动结尾）
- 推荐话题标签（5-8个）
- 封面图设计建议
- 视频脚本（分镜头/时间点）

**使用技巧：**
- 生成后可以直接编辑各个区块
- 点击「重新生成」获取不同版本
- 满意后点击「复制全文」一键复制

#### 🟡 中模式（给框架，你来填）

**适合场景：** 有想法但不知怎么展开、想保持自己的声音

**AI 产出：**
- 完整内容框架（各段主题）
- 每个段落的要点提示
- 3种不同的开头方案
- 结尾引导互动的建议

**使用技巧：**
- 先看 AI 给的框架，选择最合适的开头
- 按照框架提示填充你自己的内容
- 重点是**用你自己的语言**，AI 只是提供骨架

#### 🔵 深模式（深度分析，打造爆款）

**适合场景：** 核心内容、希望冲高互动、想打造代表作

**AI 产出：**
- 市场竞争分析（已有内容在做什么）
- 3-5 个差异化角度（你能做但别人没做的）
- 差异化理由（为什么你来做这个角度）
- 为什么现在是时机

**使用技巧：**
- 读完分析后，选择你最有感触的角度
- 结合自身真实经历来创作（AI 给角度，你给故事）
- 深模式内容建议配合更精心的封面图

### 4.2 内容精修（Refine）

觉得 AI 生成的内容有 80 分但还不够满意？用精修功能迭代到 95 分：

**快速精修指令（一键点击）：**

| 指令 | 效果 |
|------|------|
| 让语气更幽默 | 加入轻松的比喻和自嘲，降低严肃感 |
| 让开头更抓人 | 重写开头，增强悬念感或冲突感 |
| 加入真实案例 | 补充具体的数字、场景或故事 |
| 精简 30% | 删除废话，保留核心信息 |
| 适合视频号 | 调整为适合视频脚本的节奏 |
| 增强干货感 | 加入可操作的步骤或清单 |

**自定义精修：** 如果快速指令不够，在输入框写下你的具体要求：

```
示例指令：
"开头改成讲一个我在会议室里差点尿裤子的故事，
然后自然地引出时间管理的话题"

"结尾加一个问题，让读者回复他们用什么方法管理
时间，引导评论互动"
```

**版本历史：** 每次精修后，系统保留最多 3 个版本（原始版本 + 最近 2 次精修），可以随时切换回任何一个版本。

### 4.3 内容资产包

创作完成后，你拥有一套完整的**内容资产包**：

```
一篇内容的资产包：
├── 主体内容（可复制）
│   ├── 开头钩子
│   ├── 正文主体
│   └── 互动结尾
├── 话题标签（直接复制到发布平台）
├── 封面图建议（构图/文字/配色）
└── 视频脚本（分镜时间点标注）
```

> 💡 **使用技巧**：把健康度下降的时间点和你发布的商业内容对照，通常能找到明确的相关性。

## 6. 第五步：品牌匹配

### 6.1 功能说明

品牌匹配帮助你找到**真正适合你账号调性的合作品牌**，避免接了不匹配的广告影响粉丝信任。

> 📌 这是 ToB 功能，同时服务 KOC（找合适的品牌）和品牌方（找合适的 KOC）。

### 6.2 匹配逻辑

系统根据以下维度计算品牌与你的匹配度：

| 维度 | 权重 | 说明 |
|------|------|------|
| 垂类匹配 | 40% | 品牌产品类别与你的内容垂类相符 |
| 受众重叠 | 30% | 品牌目标用户与你的粉丝画像重叠 |
| 调性契合 | 20% | 品牌风格与你的内容风格相符 |
| 信任健康度 | 10% | 你当前的账号健康状态 |

### 6.3 解读匹配报告

```
┌─────────────────────────────────────────────────┐
│  🏢 XXX 效率工具品牌                              │
│                                                  │
│  匹配分：87/100  ⭐⭐⭐⭐⭐ 强烈推荐              │
│                                                  │
│  匹配理由：                                       │
│  • 效率工具与你的「职场效率」垂类高度吻合           │
│  • 目标用户均为 25-35 岁职场人群                   │
│  • 品牌倡导「系统化工作方法」与你的价值观一致        │
│                                                  │
│  合作建议：                                       │
│  • 适合「工具测评」形式（你的强项）                 │
│  • 建议在发布 3 篇常青内容后再接洽                  │
│  • 预计合理报价区间：¥XXXX-XXXX                   │
└─────────────────────────────────────────────────┘
```

### 6.4 最佳合作时机建议

系统会基于你的信任健康度，建议最佳的接广告时机：

- **健康度 ≥ 80**：信任资产充足，可以接洽合作
- **健康度 60-79**：建议先提升到 75 以上再接商业合作
- **健康度 < 60**：不建议接新的商业合作，优先修复信任资产

## 7. AI 顾问使用指南

### 7.1 什么时候用 AI 顾问？

AI 顾问是一个了解你完整账号背景的「AI 成长教练」，适合在以下场景使用：

- 📉 互动率突然下降，不知道原因
- 🤔 面对两个选题不知道选哪个
- 📅 不知道这周/这个月应该发什么类型的内容
- 💰 有品牌来询价，不知道是否应该接
- 🔄 想重新调整账号方向，需要专业建议

### 7.2 如何问出好答案？

**❌ 模糊的提问（收到通用建议）：**
```
"我怎么涨粉？"
"为什么我互动率低？"
```

**✅ 具体的提问（收到个性化建议）：**
```
"我最近连续发了 3 篇番茄工作法相关内容，
互动率从 3.5% 降到 1.8%，是同质化严重了吗？
还是应该换个话题方向？"

"有个护肤品牌来找我合作，我的垂类是职场效率，
你觉得接这个合适吗？如果接，怎么融入到我的
内容里不显得突兀？"
```

**提问框架参考：**
```
背景：[目前遇到的具体情况]
数据：[相关数据，如互动率、阅读量]
困惑：[不确定的是什么]
期望：[希望得到什么类型的建议]
```

### 7.3 多轮对话

AI 顾问支持多轮对话，记住本次会话中你说过的内容：

```
你：我最近一篇关于职场会议效率的内容只有 800 阅读

AI：这个数据比你平均低了约 40%。让我分析一下
    可能的原因...（给出分析）

你：那我下次应该怎么改标题？

AI：（基于上面的分析，给出具体标题建议）

你：能帮我改 3 个版本吗？

AI：（直接改，无需再解释背景）
```

## 8. 常见问题（FAQ）

### Q1：AI 生成的内容有版权问题吗？

**A：** 没有。AI 生成的内容是基于你的账号定位原创生成的，版权归你所有。但建议不要直接发布 AI 生成的内容，加入你的个人经历和真实感受，内容质量会更高，也更符合你的人设。

---

### Q2：为什么不同时间推荐的选题不一样？

**A：** 这是正常的。AI 在你的垂类范围内会探索不同的角度和切入点，每次推荐都略有差异。同时，热度趋势数据也在实时变化，导致蓝海分的计算结果不同。建议每次看到好选题就记录下来，积累你的内容日历。

---

### Q3：我的粉丝数超过 20000，还适合用这个工具吗？

**A：** KOC Copilot 主要针对 1000-20000 粉丝的成长期创作者优化。如果你已经超过 20000 粉，工具仍然可用，但部分推荐逻辑（如蓝海分阈值）是针对中腰部账号的，对于大账号可能需要适当调整判断标准。

---

### Q4：我可以用在多个平台吗？

**A：** 可以，但建议为每个平台单独设置账号定位。小红书和视频号的算法规则、内容节奏完全不同：
- **小红书**：图文优先，封面决定点击率，适合干货清单
- **视频号**：前 3 秒决定留存，完播率影响推流，适合故事型内容

在问卷的「内容方向」中选择你的主平台，AI 会自动调整推荐逻辑。

---

### Q5：信任健康度低于 60，我该怎么做？

**A：** 按以下步骤操作：

1. **立即停止接新的商业合作**（至少 2 周）
2. **集中发 4-6 篇高质量常青内容**（方法论/工具测评/干货清单）
3. **积极回复评论**（评论区互动是重建信任的关键）
4. **2 周后重新查看仪表盘**，分数应该回升到 65+

> ⚠️ 信任资产的修复需要时间，没有捷径，但方法是确定的：高质量内容 + 真诚互动。

---

### Q6：AI 顾问会记住我上次对话的内容吗？

**A：** 目前 AI 顾问的对话记忆**仅在本次会话内有效**，刷新页面或关闭浏览器后，对话历史会清空。但 AI 顾问始终能读取你的账号定位档案（PersonaSeed），所以每次对话它都了解你的基本背景。

**建议：** 把重要的建议截图保存，或直接复制到你的内容日历文档中。

---

### Q7：如何让推荐越来越准确？

**A：** 参与数据飞轮：

1. 用 KOC Copilot 的推荐选题创作内容
2. 发布后，把实际数据（阅读量、互动率）反馈给系统
3. 系统更新你的账号定位种子版本（v1 → v2 → v3...）
4. 下一次推荐会更精准地适应你的实际情况

**你的反馈越多，系统越懂你。**

---

### Q8：某个选题的蓝海分很高，但我不感兴趣怎么办？

**A：** 数据是参考，不是命令。KOC Copilot 的建议逻辑是：**在你的垂类内找到机会**，但你对选题的热情才是内容质量的关键。如果一个选题蓝海分高但你没有话说，可以：

1. 点击「重新推荐」寻找你也感兴趣的高蓝海选题
2. 保留这个选题，先去了解相关知识后再创作
3. 用 AI 顾问探讨"如何把这个话题跟我真实经历结合"
